# Embeddium Au Naturel

A fork of [Embeddium](https://github.com/embeddedt/embeddium), which is itself based on [Sodium](https://github.com/CaffeineMC/sodium-fabric) - a free and open-source performance mod for Minecraft clients.

This fork is **not supported by embeddedt or CaffeineMC**. Do not report issues encountered with it to their bug trackers or Discord servers.

## About This Fork

This repository is a personal fork of Embeddium for experimental modifications and customization. For the original, actively-maintained version, please visit [embeddedt/embeddium](https://github.com/embeddedt/embeddium).

## Features

* All performance improvements from Sodium 0.5.8 and earlier, including a rewritten terrain renderer and various optimizations to the immediate-mode rendering pipeline
* Available for Minecraft Forge on 1.20.1 and older, and Fabric/NeoForge on 1.20.1 and newer
* Integrated Fabric Rendering API support (Indium is not required)
* Additional APIs for mod integration
* Optional support for translucency sorting (can be enabled in Video Settings)
